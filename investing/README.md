# WP-04 · 个人投资白皮书

> 长期主义者的现金流地基、指数化引擎与资产配置纪律

**版本** v1.0 · **更新** 2026-07-06

## 摘要

为没有信息优势、没时间盯盘的普通人给出高胜率策略：先以应急金与现金流打地基，再以低成本宽基指数基金为核心引擎，辅以资产配置与再平衡，并用自动化与规则化对抗行为偏差。仅教育用途，非投资建议。

## 执行摘要

投资的胜负，很大部分在开始投之前就由**储蓄率**与**现金流地基**决定，而非选股水平。本白皮书主张一条被反复验证的路径：**低成本宽基指数 + 定投 + 长期持有 + 定期再平衡**——不追求跑赢市场，而是拿到市场平均收益并把成本、税与情绪的损耗降到最低。

我们强调优先级瀑布：先建应急金、清高息债、用足税优账户，再谈长期投资；资产配置（股/债/现金比例）决定组合大部分波动，比选具体标的更重要。真正的敌人是自己——用自动化定投与规则化再平衡替代临场情绪决策。对工程师，**投资于职业能力本身（增收）往往是回报最高的"理财"**。

## 关键要点

- 储蓄率比收益率更决定终局，是你唯一能完全掌控的变量。
- 低成本宽基指数 + 定投 + 长期持有，长期打败多数主动投资者。
- 资产配置决定组合大部分波动；先配置，再选基。
- 应急金是保险不是投资：3–6 个月开支，放货币/短债类，避免低点割肉。
- 用自动化与预设规则对抗追涨杀跌等行为偏差。
