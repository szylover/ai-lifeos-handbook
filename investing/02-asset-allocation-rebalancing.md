# 资产配置与再平衡实务

> 本文仅用于教育用途，非投资建议。本文只讨论可复现的资产配置与再平衡方法，不预测市场、不推荐具体证券、不替代持牌顾问意见。

> 研究普遍认为：**资产配置**（而非选股/择时）解释了组合绝大部分的收益波动。先把配置搭对，再谈其他。

## 本章定位：把“我该买什么”改成“我该承担什么风险”

资产配置不是挑一个“今年会涨”的资产，而是先定义：

1. 你未来什么时候需要钱；
2. 你能承受多大的账面回撤；
3. 你需要多少长期增长；
4. 你愿意为降低波动放弃多少预期收益；
5. 你如何在市场偏离目标时机械地把组合拉回轨道。

读完本章，你应该能独立完成以下动作：

- [ ] 把股票、债券、现金、另类资产分别映射到“增长、减震、流动性、低相关”角色；
- [ ] 用年龄法、风险承受力法、投资期限法给自己写出一份目标配置；
- [ ] 用相关性和组合方差公式解释“为什么多资产组合可能降低波动”；
- [ ] 设定日历再平衡、阈值再平衡、混合再平衡的触发规则；
- [ ] 用新增资金、分红、账户内转换、卖出交易四种方式完成再平衡；
- [ ] 在电子表格里按目标权重、当前市值、偏离金额计算具体买卖金额；
- [ ] 记录每次再平衡的原因、税费影响和执行结果，避免事后凭感觉改规则。

配套阅读：如果你还没有明确储蓄率、FIRE 数字和应急金，请先看 [FIRE 基础](01-fire-basics.md) 与 [应急资金与现金流管理](04-emergency-fund-cashflow.md)；如果你准备把资产配置落到指数基金，请接着看 [指数投资实操指南](03-index-investing-guide.md)。

## 1. 为什么资产配置通常比选股/择时更重要

### 1.1 先区分三种“收益来源”

| 层级 | 你在决定什么 | 典型例子 | 可控程度 | 常见误区 |
| --- | --- | --- | --- | --- |
| 资产配置 | 股票、债券、现金等大类资产比例 | 70% 股票 + 25% 债券 + 5% 现金 | 高 | 频繁因为新闻改变比例 |
| 工具选择 | 用什么产品承载资产类别 | 宽基指数基金、债券基金、货币基金 | 中 | 追逐过去一年排名 |
| 证券选择/择时 | 买哪只股票、何时进出 | 买入单一公司、预测加息后走势 | 低 | 把运气当能力 |

对长期投资者来说，资产配置决定了组合的主要风险暴露：

- 股票比例越高，长期增长潜力通常越高，但短期回撤更大；
- 债券比例越高，组合波动通常更低，但长期预期收益也可能更低；
- 现金比例越高，流动性越强，但长期购买力更容易被通胀侵蚀；
- 另类资产如果与股票/债券相关性较低，可能改善组合形态，但也可能带来流动性、估值和费用问题。

### 1.2 一个简单但有用的直觉

假设你有两个组合：

| 组合 | 股票 | 债券 | 现金 | 预期年化收益 | 可能的一年回撤 |
| --- | ---: | ---: | ---: | ---: | ---: |
| A：激进 | 90% | 10% | 0% | 较高 | 可能很深 |
| B：稳健 | 60% | 35% | 5% | 中等 | 通常较浅 |

如果未来 20 年股票表现很好，A 可能领先；如果未来 3 年遇到熊市且你必须用钱，B 的生存概率更高。资产配置的目标不是最大化纸面收益，而是在你的时间、现金流和心理约束下最大化“坚持执行的概率”。

### 1.3 资产配置的第一性原理

你要优化的不是“单个资产收益”，而是“整个家庭资产负债表的风险”。因此配置前先问四个问题：

1. **什么时候用钱？** 3 年内要用的钱不应依赖股票市场上涨。
2. **现金流是否稳定？** 工资稳定、储蓄率高的人，通常能承受更高波动。
3. **负债是否刚性？** 房贷、教育支出、赡养支出越刚性，流动性垫越重要。
4. **是否能在下跌中执行计划？** 如果 -30% 会导致你清仓，目标股票比例就设高了。

完成标准：你能用一句话说明自己的配置目标，例如：

> “我配置 75% 股票、20% 债券、5% 现金，因为投资期限超过 15 年，工资现金流稳定，但我希望组合最大回撤不要让我在熊市中停止定投。”

## 2. 四类资产的角色：不要让一个资产承担所有任务

### 2.1 股票/权益：增长引擎

| 维度 | 说明 |
| --- | --- |
| 核心功能 | 分享企业盈利增长与经济增长 |
| 主要收益来源 | 盈利增长、股息、估值变化 |
| 主要风险 | 熊市回撤、估值压缩、行业/国家集中风险 |
| 适合资金 | 5–10 年以上不用的钱 |
| 实操载体 | 宽基股票指数基金、股票 ETF、养老金权益类账户 |

股票是组合中最常见的长期增长来源，但它的“价格”是波动。用股票前要接受三件事：

- 股票可能多年不赚钱；
- 全球或单一市场指数也可能出现 30%–50% 的历史级回撤；
- 高股票比例只有在你不被迫卖出时才有意义。

### 2.2 债券/固收：减震器与现金流平滑器

| 维度 | 说明 |
| --- | --- |
| 核心功能 | 降低组合波动、提供相对稳定的利息现金流 |
| 主要收益来源 | 票息、利率变化带来的价格变化、信用利差变化 |
| 主要风险 | 利率风险、信用风险、久期风险、通胀风险 |
| 适合资金 | 中期资金、风险预算中的防守部分 |
| 实操载体 | 国债、投资级债券基金、短债基金、目标久期债券基金 |

债券不是“绝对不会亏”的资产。长期债券在利率上升时可能明显下跌，高收益债在危机中可能与股票一起下跌。实践中要把债券拆成两个维度：

1. **信用质量**：国债/政策性债/投资级债通常更防守，高收益债更像带债券外衣的风险资产；
2. **久期**：久期越长，对利率变化越敏感。

### 2.3 现金/货币基金：流动性与选择权

| 维度 | 说明 |
| --- | --- |
| 核心功能 | 应急、短期开支、等待再平衡机会 |
| 主要收益来源 | 存款利息、货币基金收益、短期票据收益 |
| 主要风险 | 通胀侵蚀、收益率下行、机会成本 |
| 适合资金 | 0–24 个月确定要用的钱、应急金 |
| 实操载体 | 活期/定期存款、货币基金、短期国债工具 |

现金的价值不是高收益，而是“不需要在坏价格卖出长期资产”。建议把现金拆成两层：

- **应急金**：通常覆盖 3–12 个月必要支出，单独管理，不参与再平衡；
- **组合现金**：作为目标配置的一部分，如 0%–10%，参与再平衡计算。

### 2.4 另类资产：低相关性的候选项，而不是魔法收益

| 类别 | 可能角色 | 关键风险 | 适合处理方式 |
| --- | --- | --- | --- |
| REITs | 房地产现金流暴露 | 利率敏感、地产周期 | 小比例、视为权益或另类 |
| 黄金/商品 | 通胀或危机情景对冲 | 长期现金流为零、波动高 | 明确上限比例 |
| 私募/风投 | 非公开市场风险溢价 | 流动性差、估值滞后、费用高 | 只用长期闲钱 |
| 加密资产 | 高波动非传统资产 | 极端波动、监管、托管风险 | 若配置，设很低上限并写明亏损承受 |

另类资产只有在满足以下条件时才值得纳入：

1. 你能解释它在组合中的角色；
2. 你知道它与股票/债券的相关性可能在危机中上升；
3. 你能承受流动性差或价格剧烈波动；
4. 你有明确的目标比例和再平衡规则。

## 3. 从零设定目标配置：三种可执行方法

### 3.1 方法 A：年龄法作为起点

常见经验法：

```text
股票比例 ≈ 110 − 年龄
债券/现金比例 ≈ 年龄 − 10
```

示例：35 岁投资者：

```text
股票比例 = 110 − 35 = 75%
债券/现金比例 = 25%
```

可落地为：

| 年龄 | 起点配置 | 适合的默认解释 |
| ---: | --- | --- |
| 25 | 85% 股票 / 10% 债券 / 5% 现金 | 投资期限长，重点是增长 |
| 35 | 75% 股票 / 20% 债券 / 5% 现金 | 增长为主，加入减震器 |
| 45 | 65% 股票 / 30% 债券 / 5% 现金 | 兼顾增长与稳定 |
| 55 | 55% 股票 / 35% 债券 / 10% 现金 | 降低退休前序列风险 |
| 65 | 40% 股票 / 45% 债券 / 15% 现金 | 现金流和回撤控制优先 |

年龄法的用法：只当“初始锚点”，不要机械照搬。软件工程师式做法是把它当默认参数，然后用约束条件覆盖。

### 3.2 方法 B：风险承受力法

用下面的压力测试选择股票比例。假设你的长期投资组合为 100 万元：

| 股票比例 | 熊市中股票下跌 40%、债券不变时的组合回撤 | 账面亏损 | 如果你会怎么做 |
| ---: | ---: | ---: | --- |
| 90% | -36% | -36 万 | 仍能按计划买入才考虑 |
| 80% | -32% | -32 万 | 需要很强现金流和纪律 |
| 70% | -28% | -28 万 | 成长型长期投资者常见 |
| 60% | -24% | -24 万 | 平衡型配置常见 |
| 40% | -16% | -16 万 | 保守型或接近用钱阶段 |

执行步骤：

1. 写下你的总投资资产，不含自住房和应急金；
2. 选择一个股票压力场景，例如 `-40%`；
3. 计算组合可能下跌：`股票比例 × 40%`；
4. 把百分比换成金额；
5. 问自己：如果明天账户少了这笔钱，我是否会卖出、停止定投或影响家庭决策？
6. 如果答案是“会”，把股票比例下调 10 个百分点，再重复。

完成标准：你能写出“我能承受的最大账面回撤金额”和对应股票比例。

### 3.3 方法 C：资金用途/期限分桶

把钱按用途分桶，再把每个桶映射到资产：

| 用途 | 时间 | 推荐资产特征 | 示例配置 |
| --- | ---: | --- | --- |
| 应急金 | 随时 | 高流动性、低波动 | 现金/货币基金 100% |
| 1–3 年支出 | 短期 | 本金稳定优先 | 现金 + 短债 |
| 3–7 年目标 | 中期 | 中低波动 | 债券为主，少量股票 |
| 7–15 年目标 | 长期 | 增长与波动平衡 | 股票 + 债券 |
| 15 年以上退休 | 超长期 | 增长优先 | 股票为主 |

示例：

| 资金 | 金额 | 期限 | 配置 |
| --- | ---: | ---: | --- |
| 应急金 | ¥120,000 | 随时 | 100% 现金，独立于投资组合 |
| 3 年后买车 | ¥100,000 | 3 年 | 70% 现金 / 30% 短债 |
| 退休账户 | ¥800,000 | 20 年 | 80% 股票 / 15% 债券 / 5% 现金 |

这种方法的优点是减少“把短期钱拿去赌长期收益”的错误。

### 3.4 把三种方法合并成一份投资政策声明（IPS）

建议写一份 1 页的 Investment Policy Statement：

```markdown
# 我的资产配置政策

目标：长期财富增长，并控制单年极端回撤对家庭现金流的影响。
范围：只包含长期投资账户，不包含应急金和 3 年内确定支出。
目标配置：股票 75%，债券 20%，现金 5%。
股票内部：全球/本国宽基为主，不做单股集中押注。
债券内部：以高信用质量中短久期为主。
再平衡规则：每季度检查；任一大类资产偏离目标绝对值超过 5 个百分点时执行。
税务原则：优先使用新增资金、分红和税延账户；应税账户卖出前估算税费。
禁止事项：不因新闻、社交媒体观点、短期涨跌改变目标配置。
复盘频率：每年 1 月复核收入、支出、目标和风险承受力。
```

完成标准：目标配置不是一句“我想稳一点”，而是一套可执行规则。

## 4. 有效前沿直觉：不是追求最高收益，而是寻找同等风险下更高收益

### 4.1 预期收益的组合公式

如果组合包含 $n$ 类资产，第 $i$ 类资产权重为 $w_i$，预期收益为 $E(R_i)$，组合预期收益为：

$$
E(R_p)=\sum_{i=1}^{n}w_iE(R_i)
$$

约束条件通常是：

$$
\sum_{i=1}^{n}w_i=1
$$

直觉：组合收益是各资产预期收益的加权平均。提高股票权重通常提高预期收益，但也提高波动。

### 4.2 组合方差：相关性决定“混合后有多抖”

组合风险不是各资产波动的简单加权平均，因为资产之间会一起涨跌或反向波动。组合方差可写为：

$$
\sigma_p^2 = \sum_i \sum_j w_i w_j \sigma_{ij}
$$

题目中常见简写为：

$$
\sigma_p^2 = \sum w_i w_j \sigma_{ij}
$$

其中：

- $w_i$、$w_j$ 是资产权重；
- $\sigma_{ij}$ 是资产 $i$ 与资产 $j$ 的协方差；
- 当 $i=j$ 时，$\sigma_{ii}$ 就是资产 $i$ 的方差；
- 协方差越低，组合越可能降低波动。

### 4.3 两资产组合的可手算版本

两个资产 A、B 的组合方差：

$$
\sigma_p^2=w_A^2\sigma_A^2+w_B^2\sigma_B^2+2w_Aw_B\rho_{AB}\sigma_A\sigma_B
$$

假设：

| 参数 | 数值 |
| --- | ---: |
| 股票预期收益 | 8% |
| 债券预期收益 | 3% |
| 股票波动率 $\sigma_A$ | 18% |
| 债券波动率 $\sigma_B$ | 6% |
| 股票/债券相关系数 $\rho$ | 0.2 |
| 权重 | 股票 70% / 债券 30% |

预期收益：

$$
E(R_p)=0.7\times8\%+0.3\times3\%=6.5\%
$$

方差：

$$
\sigma_p^2=0.7^2\times0.18^2+0.3^2\times0.06^2+2\times0.7\times0.3\times0.2\times0.18\times0.06
$$

逐项计算：

| 项 | 计算 | 结果 |
| --- | --- | ---: |
| 股票自身风险 | $0.49\times0.0324$ | 0.015876 |
| 债券自身风险 | $0.09\times0.0036$ | 0.000324 |
| 协方差项 | $2\times0.7\times0.3\times0.2\times0.18\times0.06$ | 0.0009072 |
| 合计方差 | 三项相加 | 0.0171072 |
| 组合波动率 | $\sqrt{0.0171072}$ | 13.08% |

如果只按权重线性平均波动率：

```text
0.7 × 18% + 0.3 × 6% = 14.4%
```

实际组合波动率约 13.08%，低于 14.4%，这就是低相关资产带来的分散化效果。

### 4.4 有效前沿的工程化理解

把每一种权重组合看成一个参数配置：

```text
输入：资产预期收益、波动率、相关性矩阵、权重约束
输出：组合预期收益、组合波动率
目标：在同等波动率下找更高预期收益，或在同等预期收益下找更低波动率
```

有效前沿不是告诉你“唯一正确组合”，而是帮你避免明显低效组合。例如：

| 组合 | 预期收益 | 波动率 | 判断 |
| --- | ---: | ---: | --- |
| X | 6.0% | 14% | 可接受 |
| Y | 6.0% | 18% | 被 X 支配：同收益更高风险 |
| Z | 6.8% | 15% | 若能承受多 1% 波动，可考虑 |

实践注意：预期收益和相关性来自历史或假设，未来会变化。所以普通投资者不必追求数学上最优，而是用有效前沿建立直觉：**相关性低、成本低、纪律强的组合，通常比押注单一资产更稳健。**

## 5. 分散化数学：你到底在分散什么

### 5.1 分散化不是“买很多只”

如果你买了 20 只同一行业股票，可能仍然高度集中。有效分散至少包含四层：

| 层级 | 需要检查的问题 | 例子 |
| --- | --- | --- |
| 资产类别 | 是否只有股票，没有债券/现金？ | 100% 科技股基金 |
| 地域/货币 | 是否只押注单一国家/货币？ | 全部本币资产 |
| 行业/风格 | 是否集中在科技、成长、小盘？ | 只买 AI 主题基金 |
| 产品/机构 | 是否集中在单一平台或单一管理人？ | 所有现金放同一机构 |

### 5.2 相关性矩阵的最小表格

在电子表格中可以维护一个简化相关性矩阵：

| 相关性 | 股票 | 债券 | 现金 | 另类 |
| --- | ---: | ---: | ---: | ---: |
| 股票 | 1.0 | 0.2 | 0.0 | 0.5 |
| 债券 | 0.2 | 1.0 | 0.1 | 0.1 |
| 现金 | 0.0 | 0.1 | 1.0 | 0.0 |
| 另类 | 0.5 | 0.1 | 0.0 | 1.0 |

使用原则：

- 相关性接近 1：多数时间同涨同跌，分散化有限；
- 相关性接近 0：走势关系弱，可能降低组合波动；
- 相关性为负：可能提供更强减震，但现实中不稳定；
- 危机时期相关性可能上升，不能只看平静期数据。

### 5.3 集中度检查：Herfindahl-Hirschman Index（HHI）

可以用 HHI 粗略衡量权重集中度：

$$
HHI=\sum_{i=1}^{n}w_i^2
$$

示例：

| 组合 | 权重 | HHI | 直觉 |
| --- | --- | ---: | --- |
| 组合 A | 100% 单一资产 | 1.00 | 极度集中 |
| 组合 B | 70/30 | 0.58 | 两资产，仍偏集中 |
| 组合 C | 50/30/15/5 | 0.365 | 更分散 |
| 组合 D | 25/25/25/25 | 0.25 | 四资产等权 |

HHI 不是投资优化工具，但它能快速提醒你：名义上“买了很多产品”，实际可能仍由一个风险因子主导。

## 6. 再平衡：把组合风险拉回目标，而不是预测涨跌

### 6.1 再平衡的定义

再平衡 = 当实际配置偏离目标配置时，通过买入低配资产、卖出高配资产、使用新增资金或调整未来买入方向，使组合回到预设目标或目标区间。

它的核心目的：

1. **风险控制**：股票涨多后比例过高，组合风险超过计划；
2. **纪律执行**：不靠情绪判断，靠规则行动；
3. **被动低买高卖**：卖出涨多的，买入相对落后的；
4. **防止漂移**：避免几年后组合变成完全不同的风险形态。

### 6.2 再平衡不保证提高收益

再平衡可能提高风险调整后收益，也可能在单边趋势市场中拖累收益。例如股票连续上涨 10 年，频繁卖股票买债券会降低绝对收益。但它仍有价值，因为目标是把组合保持在你能承受的风险水平。

正确评价标准不是“这次卖早了吗”，而是：

- 执行后组合是否回到目标风险；
- 交易成本和税费是否可接受；
- 是否遵守事先写下的规则；
- 是否避免了情绪化追涨杀跌。

## 7. 三类再平衡方法

### 7.1 方法一：日历再平衡（Calendar Rebalancing）

规则：在固定日期检查并再平衡，例如每季度末、每半年、每年 1 月第一个周末。

| 检查频率 | 适合人群 | 优点 | 缺点 |
| --- | --- | --- | --- |
| 每月 | 现金流频繁、组合复杂 | 偏离发现快 | 容易过度交易 |
| 每季度 | 多账户、多产品 | 平衡执行成本和纪律 | 需要固定流程 |
| 每半年 | 长期指数投资者 | 简洁 | 极端行情中偏离较久 |
| 每年 | 极简投资者 | 最省心 | 可能错过大幅漂移 |

日历法步骤：

1. 在日历里创建固定事件：`每季度最后一个周六 30 分钟`；
2. 打开总资产表，更新所有资产市值；
3. 计算实际权重；
4. 判断是否执行买卖；
5. 记录“检查但未交易”或“检查并交易”。

完成标准：即使不交易，也留下检查记录，避免未来凭记忆复盘。

### 7.2 方法二：阈值/带宽再平衡（Threshold/Band Rebalancing）

规则：只有当资产偏离目标超过设定带宽时才动作。

常见带宽：

| 目标权重 | 绝对带宽 | 触发区间 | 例子 |
| ---: | ---: | --- | --- |
| 70% 股票 | ±5 个百分点 | <65% 或 >75% | 适合大类资产 |
| 20% 债券 | ±5 个百分点 | <15% 或 >25% | 适合中等权重资产 |
| 5% 现金 | ±2 个百分点 | <3% 或 >7% | 小权重资产可用较小绝对带宽 |

也可用相对带宽：

```text
触发阈值 = 目标权重 × 20%
目标 70% 股票 → 带宽 14 个百分点 → 56%–84%
目标 5% 现金 → 带宽 1 个百分点 → 4%–6%
```

实践中更常用混合规则：

```text
每季度检查一次；
若任一大类资产偏离目标超过 ±5 个百分点，执行再平衡；
若偏离未超过阈值，只把新增资金投向低配资产。
```

### 7.3 方法三：现金流再平衡

优先用“流入/流出”修正偏离，减少卖出：

| 现金流来源 | 用法 |
| --- | --- |
| 每月定投 | 全部投向低配资产，直到回到目标区间 |
| 分红/利息 | 不自动再投资到原资产，而是流向低配资产 |
| 年终奖/奖金 | 一次性补足低配资产 |
| 退休提款 | 优先从超配资产卖出，顺便再平衡 |

现金流再平衡适合积累期投资者。它的核心优势是降低交易次数、税费和心理摩擦。

## 8. 税务友好的再平衡顺序

税务规则因地区、账户类型和产品而异；这里给通用执行框架。具体税率与申报要求请以所在地法律和专业意见为准。

### 8.1 推荐顺序

| 优先级 | 动作 | 为什么 |
| ---: | --- | --- |
| 1 | 用新增资金买入低配资产 | 不触发卖出税费 |
| 2 | 改变未来定投方向 | 无交易成本或成本低 |
| 3 | 用分红/利息买入低配资产 | 避免自动买入已超配资产 |
| 4 | 在税延/免税账户内转换 | 可能不触发当期资本利得税 |
| 5 | 在应税账户卖出亏损资产 | 可能抵税，需遵守当地规则 |
| 6 | 在应税账户卖出盈利资产 | 最后手段，先估算税费 |

### 8.2 卖出前的税费检查

卖出前填写一张小表：

| 字段 | 示例 | 说明 |
| --- | ---: | --- |
| 当前市值 | ¥180,000 | 准备卖出的资产市值 |
| 成本基础 | ¥120,000 | 买入成本或税务成本 |
| 未实现收益 | ¥60,000 | 当前市值 − 成本基础 |
| 预计税率 | 20% | 按所在地规则估算 |
| 预计税额 | ¥12,000 | 未实现收益 × 税率 |
| 交易费/滑点 | ¥100 | 券商佣金、申赎费、价差 |
| 税后可用金额 | ¥167,900 | 市值 − 税额 − 费用 |

如果卖出会产生很高税费，可以考虑：

- 只再平衡到目标区间边缘，而不是精确目标；
- 分多期卖出；
- 用未来新增资金慢慢修正；
- 在税延账户里做相反方向调整；
- 捐赠或转移高盈利资产（如当地规则允许）。

### 8.3 资产位置（Asset Location）

资产配置回答“买多少”，资产位置回答“放哪里”。通用原则：

| 资产 | 更适合账户 | 原因 |
| --- | --- | --- |
| 高频派息债券 | 税延/免税账户 | 利息可能按普通收入征税 |
| 高换手主动基金 | 税延/免税账户 | 降低年度税务拖累 |
| 宽基股票指数 | 应税账户也可 | 换手低、税务效率较高 |
| 现金/货基 | 按流动性优先 | 应急可取比税务优化更重要 |

执行时不要为了税务最优牺牲风险控制。税务是约束条件，不是资产配置的唯一目标。

## 9. 完整实操示例：从漂移到再平衡

### 9.1 背景设定

投资者小林的目标配置：

| 资产类别 | 目标权重 | 代表产品 | 备注 |
| --- | ---: | --- | --- |
| 股票 | 70% | 宽基股票指数基金 | 长期增长 |
| 债券 | 25% | 中短债基金 | 降低波动 |
| 现金 | 5% | 货币基金 | 组合现金，不含应急金 |

再平衡规则：

```text
每季度检查；
任一资产类别偏离目标超过 ±5 个百分点时再平衡；
优先使用账户内现金和新增资金；
应税账户卖出前估算税费。
```

### 9.2 当前持仓：市场上涨后股票超配

| 资产类别 | 当前份额/单位 | 当前价格 | 当前市值 | 目标权重 | 当前权重 | 偏离 |
| --- | ---: | ---: | ---: | ---: | ---: | ---: |
| 股票基金 | 8,000 份 | ¥125.00 | ¥1,000,000 | 70% | 80.00% | +10.00% |
| 债券基金 | 2,500 份 | ¥96.00 | ¥240,000 | 25% | 19.20% | -5.80% |
| 现金/货基 | 10,000 份 | ¥1.00 | ¥10,000 | 5% | 0.80% | -4.20% |
| **合计** |  |  | **¥1,250,000** | 100% | 100.00% |  |

触发判断：

- 股票偏离 `+10.00%`，超过 `+5%` 阈值；
- 债券偏离 `-5.80%`，超过 `-5%` 阈值；
- 需要执行再平衡。

### 9.3 计算目标市值

总市值：

```text
¥1,000,000 + ¥240,000 + ¥10,000 = ¥1,250,000
```

目标市值：

| 资产类别 | 总市值 | 目标权重 | 目标市值 |
| --- | ---: | ---: | ---: |
| 股票 | ¥1,250,000 | 70% | ¥875,000 |
| 债券 | ¥1,250,000 | 25% | ¥312,500 |
| 现金 | ¥1,250,000 | 5% | ¥62,500 |
| **合计** |  | 100% | **¥1,250,000** |

### 9.4 计算需要买卖的金额

| 资产类别 | 当前市值 | 目标市值 | 差额 = 目标 − 当前 | 动作 |
| --- | ---: | ---: | ---: | --- |
| 股票 | ¥1,000,000 | ¥875,000 | -¥125,000 | 卖出 |
| 债券 | ¥240,000 | ¥312,500 | +¥72,500 | 买入 |
| 现金 | ¥10,000 | ¥62,500 | +¥52,500 | 增加 |
| **合计** | ¥1,250,000 | ¥1,250,000 | ¥0 | 资金守恒 |

解释：卖出股票 ¥125,000，其中 ¥72,500 买债券，¥52,500 留作现金/货基。

### 9.5 计算份额变化

| 资产类别 | 动作金额 | 当前价格 | 份额变化公式 | 份额变化 |
| --- | ---: | ---: | --- | ---: |
| 股票基金 | -¥125,000 | ¥125.00 | `125000 / 125` | 卖出 1,000 份 |
| 债券基金 | +¥72,500 | ¥96.00 | `72500 / 96` | 买入 755.2083 份 |
| 现金/货基 | +¥52,500 | ¥1.00 | `52500 / 1` | 增加 52,500 份 |

如果平台只支持整数份额或按金额申购：

- 股票卖出 1,000 份；
- 债券按金额申购 ¥72,500；
- 剩余资金进入货币基金；
- 因四舍五入造成的几十元偏差，下次检查时再处理。

### 9.6 再平衡后持仓

| 资产类别 | 原份额 | 份额变化 | 新份额 | 价格 | 新市值 | 新权重 |
| --- | ---: | ---: | ---: | ---: | ---: | ---: |
| 股票基金 | 8,000.0000 | -1,000.0000 | 7,000.0000 | ¥125.00 | ¥875,000 | 70.00% |
| 债券基金 | 2,500.0000 | +755.2083 | 3,255.2083 | ¥96.00 | ¥312,500 | 25.00% |
| 现金/货基 | 10,000.0000 | +52,500.0000 | 62,500.0000 | ¥1.00 | ¥62,500 | 5.00% |
| **合计** |  |  |  |  | **¥1,250,000** | 100.00% |

### 9.7 税务版本：如果卖股票会触发税费

假设股票基金在应税账户，卖出 ¥125,000 中有 40% 是未实现收益：

| 项目 | 数值 |
| --- | ---: |
| 卖出市值 | ¥125,000 |
| 未实现收益比例 | 40% |
| 应税收益 | ¥50,000 |
| 资本利得税率假设 | 20% |
| 预计税额 | ¥10,000 |
| 税后可用于买债/现金 | ¥115,000 |

此时不能机械照搬税前表格。可选择两种方案：

**方案 A：仍卖出 ¥125,000，税款从现金外部支付**

| 资产 | 再平衡后 | 说明 |
| --- | ---: | --- |
| 股票 | ¥875,000 | 精确回到 70% |
| 债券 | ¥312,500 | 精确回到 25% |
| 现金 | ¥62,500 | 精确回到 5% |
| 外部现金 | -¥10,000 | 用于缴税 |

**方案 B：只卖出到阈值内，减少税费**

目标不是 70%，而是先把股票从 80% 降到 75% 阈值上沿：

```text
股票目标市值 = 1,250,000 × 75% = 937,500
需要卖出 = 1,000,000 − 937,500 = 62,500
```

| 资产类别 | 当前市值 | 调整后市值 | 调整后权重 |
| --- | ---: | ---: | ---: |
| 股票 | ¥1,000,000 | ¥937,500 | 75.00% |
| 债券 | ¥240,000 | ¥282,500 | 22.60% |
| 现金 | ¥10,000 | ¥30,000 | 2.40% |
| 税费/费用 |  | 视规则估算 |  |

方案 B 没有精确回到目标，但把股票拉回允许区间，可作为税务成本较高时的折中。

## 11. 可复现再平衡流程：30 分钟版本

### 11.1 准备阶段：5 分钟

1. 打开你的资产表；
2. 确认目标配置没有临时改动；
3. 确认应急金不参与本次再平衡；
4. 确认本次检查日期与上次执行记录。

完成标准：你知道本次是在执行规则，而不是因为行情刺激临时操作。

### 11.2 数据更新：10 分钟

逐项更新：

- 每个账户的份额；
- 每个产品的最新价格或净值；
- 现金余额；
- 未到账分红、在途交易；
- 新增资金或即将提款金额。

完成标准：总市值能与券商/银行账户合计数对上，误差只来自四舍五入或在途交易。

### 11.3 触发判断：5 分钟

按表格输出检查：

```text
如果 ABS(当前权重 − 目标权重) > 阈值：触发
否则：不交易，只记录检查
```

如果没有触发：

- 本次记录为“检查，未交易”；
- 下一笔新增资金投向低配资产；
- 不因为“差一点触发”提前交易。

### 11.4 交易设计：5 分钟

按顺序选择资金来源：

1. 新增现金；
2. 分红/利息；
3. 税延账户内部转换；
4. 应税账户卖出亏损资产；
5. 应税账户卖出盈利资产。

完成标准：你能写出每笔交易的金额、份额、账户和税费备注。

### 11.5 执行与记录：5 分钟

执行后立刻记录：

- 下单时间；
- 买卖方向；
- 成交价格或申购金额；
- 费用；
- 税费估算；
- 执行后目标权重。

完成标准：未来复盘时，不需要翻聊天记录或凭记忆解释为什么交易。

```text
开始
  ↓
是否到达检查日期？
  ├─ 否：是否有大额现金流？
  │     ├─ 否：不操作
  │     └─ 是：把新增资金投向低配资产
  └─ 是：更新所有资产市值
          ↓
      任一资产偏离超过阈值？
          ├─ 否：记录“检查未交易”
          └─ 是：是否能用新增资金/分红修正？
                  ├─ 是：买入低配资产
                  └─ 否：是否能在税延账户调整？
                          ├─ 是：账户内转换
                          └─ 否：估算应税账户卖出税费后执行或分期执行
```

## 13. 常见配置组合与适用场景

以下只是教育示例，不代表推荐：

| 配置 | 适用约束 | 可能优点 | 主要代价 |
| --- | --- | --- | --- |
| 90/10/0 股票/债券/现金 | 超长期、现金流强、心理承受力高 | 长期增长潜力高 | 回撤深，执行难 |
| 80/15/5 | 年轻积累期、已有应急金 | 增长为主，保留少量防守 | 熊市仍可能大跌 |
| 70/25/5 | 长期平衡型 | 增长与减震折中 | 牛市可能跑输高股票组合 |
| 60/35/5 | 中等风险承受力 | 波动较低 | 长期预期收益较低 |
| 40/45/15 | 接近退休或提款期 | 现金流稳定性较高 | 通胀和低收益风险 |

选择组合时，不要只看收益模拟；同时做三个压力测试：

1. 股票下跌 40%；
2. 债券因利率上升下跌 10%；
3. 你失业 6 个月且需要动用现金。

## 14. 常见错误与排查表

| 错误 | 症状 | 直接后果 | 修正方法 |
| --- | --- | --- | --- |
| 追涨杀跌 | 看到上涨就提高股票比例，下跌就降低 | 高买低卖 | 只在年度复核时改目标配置 |
| 过度再平衡 | 每周都交易 | 费用、税费、精力消耗 | 设置日历 + 阈值双条件 |
| 忽略整体账户 | 只看单个券商账户 | 实际风险暴露错误 | 汇总家庭全部投资账户 |
| 把应急金投入股票 | 熊市叠加失业时被迫卖出 | 永久损失风险 | 应急金单独放现金/货基 |
| 债券买得太激进 | 高收益债在危机中与股票一起跌 | 防守失效 | 区分信用风险和久期风险 |
| 另类资产无上限 | 小仓位涨成大仓位 | 组合风险失控 | 给另类资产写上限和再平衡规则 |
| 税前精确、税后错误 | 卖出后发现税款占用现金 | 配置不达标 | 卖出前估算税费和费用 |
| 频繁修改目标 | 每次市场变化都改 IPS | 策略不可复盘 | 目标配置只在生活条件变化时复核 |

## 15. 年度复核：什么时候可以改目标配置

再平衡不是改变目标配置。只有以下情况才考虑修改目标配置：

| 触发事件 | 可调整方向 |
| --- | --- |
| 年龄增长、距离退休更近 | 逐步降低股票比例或增加现金流资产 |
| 收入稳定性显著下降 | 降低风险资产比例，提高现金垫 |
| 支出责任增加 | 增加现金和短债比例 |
| 净资产显著增长且目标已接近 | 从“最大化增长”转向“目标达成概率” |
| 长期目标改变 | 重新分桶，调整每个桶的风险 |

不建议因为以下原因改目标配置：

- 某位博主说未来要大涨/大跌；
- 过去 3 个月某资产表现最好；
- 你刚经历一次亏损，想“等回本再说”；
- 你想通过临时提高风险弥补之前错过的行情。

## 16. 可勾选执行清单

### 16.1 首次建立配置

- [ ] 写下投资范围：哪些账户和资产计入长期投资组合；
- [ ] 单独列出应急金，不把它混入风险资产；
- [ ] 写下目标配置，例如股票 70% / 债券 25% / 现金 5%；
- [ ] 写下每类资产的角色：增长、减震、流动性、低相关；
- [ ] 做一次股票 -40% 压力测试并记录金额；
- [ ] 确定再平衡频率：月度、季度、半年或年度；
- [ ] 确定阈值：例如大类资产 ±5 个百分点；
- [ ] 确定税务顺序：新增资金 → 分红 → 税延账户 → 应税账户；
- [ ] 建立资产表和交易记录表；
- [ ] 在日历里创建下一次检查提醒。

### 16.2 每次再平衡

- [ ] 更新所有账户份额、价格、现金；
- [ ] 核对总市值与账户合计；
- [ ] 计算当前权重和偏离；
- [ ] 判断是否超过阈值；
- [ ] 若未触发，记录“检查未交易”；
- [ ] 若触发，先用新增资金和分红修正；
- [ ] 卖出前估算税费、费用和滑点；
- [ ] 计算买卖金额和份额；
- [ ] 执行交易后更新表格；
- [ ] 记录本次触发原因、执行金额、执行后权重；
- [ ] 设置下一次检查日期。

### 16.3 年度复盘

- [ ] 收入稳定性是否变化；
- [ ] 支出责任是否变化；
- [ ] 投资期限是否缩短；
- [ ] 最大可承受回撤金额是否变化；
- [ ] 是否需要调整股票/债券/现金比例；
- [ ] 是否有资产位置或税务优化空间；
- [ ] 是否存在单一资产、行业、国家、平台集中风险；
- [ ] 是否仍能用一句话解释目标配置。

## 17. 练习：把本章变成你的配置系统

### 练习 A：15 分钟压力测试

产出物：一段文字 + 一张小表。

1. 写下长期投资组合总额；
2. 假设股票部分下跌 40%；
3. 计算不同股票比例下的账面亏损；
4. 选择你能承受的最高股票比例；
5. 写下原因。

模板：

| 股票比例 | 组合总额 | 股票下跌假设 | 组合亏损 | 金额 |
| ---: | ---: | ---: | ---: | ---: |
| 80% | ¥____ | -40% | -32% | ¥____ |
| 70% | ¥____ | -40% | -28% | ¥____ |
| 60% | ¥____ | -40% | -24% | ¥____ |

### 练习 B：建立再平衡表

产出物：一个电子表格，至少包含第 10 节的 A–P 列。

验收标准：

- 修改任意资产价格后，当前权重自动变化；
- 偏离超过阈值时，`是否触发` 自动变为 TRUE；
- `需买卖金额` 合计为 0 或接近 0；
- 能输出具体买卖金额和份额变化。

### 练习 C：写一份 IPS

产出物：不超过 300 字的投资政策声明。

必须包含：

- 投资范围；
- 目标配置；
- 再平衡频率和阈值；
- 税务顺序；
- 禁止事项；
- 年度复核日期。
