# 心态、身份与关系：可持续地做自由的人

> 逃离坐班不是把工牌摘下来，而是把身份、关系、节奏、现金流和意义感重新装配成一个能长期运行的生活系统。

## 本章定位：自由之后，如何不把自己弄丢

如果说 [安全退出与过渡期设计](01-safe-exit-transition.md) 解决的是“什么时候走、怎么走”，[一个人的生产力系统](03-solo-productivity-system.md) 解决的是“没人管时怎么做事”，本章解决的是更底层的问题：

**当公司、头衔、固定同事、固定作息和外部评价都变弱之后，你如何仍然稳定、健康、有连接、有方向地生活。**

这不是鸡汤章节。

它是一套心理与关系层面的操作系统：

- 用价值观和作品替代“公司名片”来定义自己。
- 用固定连接机制对抗远程/自由职业的孤独。
- 用可复用话术处理父母、亲友、社会对“不稳定”的质疑。
- 用现金流、安全垫和认知重构降低不确定性焦虑。
- 用生活设计回答“自由是为了什么”。
- 用早期预警信号保护心理健康。
- 用关系协议让伴侣、家庭、迁移和独立工作可以共存。

本章与总纲 [《逃离坐班白皮书》](../00-manifesto.md) 的关系是：总纲讲“为什么要逃离”，本章讲“逃离之后如何不坠落”。

## 读完本章，你要产出的 8 个具体文件

请不要只阅读。

读完后，你应该在自己的笔记系统里产出以下 8 个文件或页面：

1. `identity.md`：一份不依赖公司和职级的身份说明。
2. `values.md`：5 个核心价值观，以及每个价值观对应的行为证据。
3. `connection-plan.md`：每周连接计划，包括朋友、同行、社群和家人。
4. `family-script.md`：向父母/亲友解释自由职业或远程生活的版本化话术。
5. `fear-log.md`：焦虑、比较、冒名顶替感的记录与重构表。
6. `purpose-map.md`：自由到底服务哪些人生目标。
7. `mental-health-guardrails.md`：心理健康预警信号和求助清单。
8. `90-day-plan.md`：90 天可持续独立生活建设计划。

完成标准很简单：

- 每个文件至少有一页内容。
- 每个文件都有下一步行动。
- 每个文件都能在你焦虑、孤独、被质疑或失去方向时直接打开使用。

## 免责声明：本章不是心理治疗

本章提供的是自我管理、关系沟通和生活设计工具。

如果你已经出现持续失眠、明显抑郁、惊恐发作、自伤念头、长期无法工作、物质滥用、严重关系冲突等情况，请优先寻求专业帮助。

可选路径包括：

- 精神科或心理科门诊。
- 持证心理咨询师。
- 当地危机干预热线。
- 可信任家人或朋友的陪同就医。
- 公司 EAP、商业保险心理服务、学校/社区资源。

自由不是硬扛。

真正可持续的自由包括：知道什么时候要靠系统，什么时候要靠别人，什么时候要靠专业人士。

## 一、先建立一个心智模型：自由生活是一套分布式系统

资深工程师可以把自由生活理解成一套分布式系统。

坐班公司像一个单体应用：

| 公司提供的东西 | 离开后会发生什么 | 你需要自己重建什么 |
|---|---|---|
| 头衔与组织背书 | “我是谁”变模糊 | 身份系统 |
| 固定同事与会议 | 社交频率下降 | 连接系统 |
| 月薪与绩效周期 | 收入波动变大 | 财务安全系统 |
| 上级与路线图 | 优先级不再外包 | 目标系统 |
| 办公室与通勤 | 边界变弱 | 节奏系统 |
| 晋升与评价 | 状态反馈减少 | 自我评估系统 |
| HR/同事观察 | 心理风险更隐蔽 | 健康预警系统 |

逃离坐班的早期幻觉是：

> 只要没有老板、没有通勤、没有打卡，我就自由了。

真实情况通常是：

> 你移除了约束，也移除了很多隐形支架。

所以本章的目标不是让你“更坚强”，而是让你搭建新的支架。

### 自由生活的 6 个支架

1. **身份支架**：我不是某公司某职级，我是一个能创造某类价值的人。
2. **关系支架**：我不是孤岛，我有固定、低摩擦、可持续的连接。
3. **财务支架**：我不是每天靠意志对抗焦虑，我有安全垫和现金流计划。
4. **节奏支架**：我不是随时工作/随时内疚，我有日程边界。
5. **意义支架**：我不是只是在逃离，我知道自由服务什么。
6. **健康支架**：我不是等崩溃后再处理，我有早期预警和求助路径。

这 6 个支架缺一两个，你还能跑。

缺三四个，你会进入“自由但不稳定”的状态：白天拖延，晚上焦虑，收入稍微波动就恐慌，看见同龄人升职就自我怀疑，和父母聊天像答辩。

## 二、身份重建：离开公司名片后，你到底是谁

很多工程师的身份是这样构成的：

```text
我是谁 = 公司品牌 + 职级 + 技术栈 + 薪资区间 + 同事认可 + 朋友圈可展示的项目
```

这套身份在坐班体系里有效。

问题是，一旦你离开大厂、外企、明星团队或稳定岗位，它会突然失效。

你可能会遇到这些感受：

- 自我介绍时不知道怎么说。
- 不想参加同学聚会，因为别人会问“你现在在哪家公司”。
- 看到前同事升职、跳槽、发工牌，心里有刺痛感。
- 接项目时不敢报价，因为没有公司背书。
- 明明能力还在，却觉得“我是不是掉队了”。

这不是你脆弱。

这是身份系统过度依赖外部容器。

### 2.1 把身份从“所属”改成“能力 + 价值 + 作品”

旧身份格式：

```text
我是某公司 P/T/IC/Manager，做某业务。
```

新身份格式：

```text
我帮助某类人，用某种能力，解决某类问题，并通过作品/服务/产品持续证明。
```

示例：

| 旧身份 | 新身份 |
|---|---|
| 我是某大厂后端工程师 | 我帮助增长期团队设计可扩展的后端系统和 AI 工作流 |
| 我是前端 TL | 我帮助独立产品和 B2B SaaS 把复杂功能做成可用界面 |
| 我是算法工程师 | 我帮助业务团队把 LLM/RAG 从 demo 落到可评估、可维护的生产系统 |
| 我是全栈工程师 | 我用工程、产品和 AI 自动化，把模糊需求变成可上线的软件 |
| 我是自由职业者 | 我为中小团队提供可交付、可维护、可复用的软件系统建设服务 |

请注意：新身份不要求你马上很成功。

它要求你从“被组织定义”切换为“被问题、能力和作品定义”。

### 2.2 练习：写出你的三层身份

打开 `identity.md`，写下三层身份。

#### 第一层：事实身份

格式：

```text
我过去主要做过：
- 行业：
- 技术栈：
- 项目类型：
- 用户/客户类型：
- 我解决过的高频问题：
```

示例：

```text
我过去主要做过：
- 行业：跨境电商、B2B SaaS、内部平台
- 技术栈：TypeScript、React、Node.js、PostgreSQL、LLM API
- 项目类型：后台系统、数据看板、AI 助手、自动化工具
- 用户/客户类型：运营、销售、客服、创始人
- 我解决过的高频问题：把混乱流程产品化，把重复工作自动化，把数据入口统一
```

完成标准：不少于 20 条事实，不使用“优秀”“负责”“资深”这类形容词。

#### 第二层：价值身份

格式：

```text
我在市场中创造的价值是：
- 帮谁：
- 从什么痛苦状态：
- 到什么更好状态：
- 通过什么方法：
- 如何证明：
```

示例：

```text
我在市场中创造的价值是：
- 帮谁：10–100 人的 SaaS/电商团队
- 从什么痛苦状态：需求多、流程乱、重复劳动高、AI 工具停留在试用
- 到什么更好状态：核心流程可视化、自动化、可监控、可迭代
- 通过什么方法：全栈开发、流程梳理、AI workflow、数据看板
- 如何证明：案例文章、demo、客户推荐、复购、可量化节省工时
```

完成标准：别人读完后能知道你适合接什么活、不适合接什么活。

#### 第三层：存在身份

这一层不谈市场。

它回答：即使没有公司、没有 title、没有收入高峰，我仍然是谁。

模板：

```text
即使外部标签变化，我仍然是一个：
- 会持续学习的人，因为：
- 会解决复杂问题的人，因为：
- 重视自由/诚实/创造/家庭/健康的人，因为：
- 愿意为长期主义付出代价的人，因为：
```

示例：

```text
即使外部标签变化，我仍然是一个：
- 会持续学习的人，因为我每年都会系统掌握一个新领域，并产出教程或项目
- 会解决复杂问题的人，因为我能把混乱需求拆成可执行任务并交付
- 重视自由与诚实的人，因为我不愿用长期消耗换短期体面
- 愿意为长期主义付出代价的人，因为我可以接受 6–12 个月过渡期的慢增长
```

完成标准：这段文字在你低谷时读起来仍然站得住，不像自我催眠。

### 2.3 价值观不是口号，而是决策函数

自由生活会不断逼你做选择：

- 要不要接高价但消耗人的客户？
- 要不要为了现金流回去全职？
- 要不要搬去更便宜但社交更少的城市？
- 要不要做短期流量内容？
- 要不要接受亲友介绍的“稳定工作”？

没有价值观，你会每次都重新纠结。

价值观的作用是降低决策成本。

#### 价值观提取法：峰值/低谷/嫉妒/愤怒

在 `values.md` 写 4 组问题。

**A. 峰值时刻**

```text
过去 5 年，我最有活力的 3 个时刻是什么？
当时我在做什么？
我和谁在一起？
我解决了什么问题？
这个时刻满足了我的什么需求？
```

**B. 低谷时刻**

```text
过去 5 年，我最耗竭的 3 个时刻是什么？
当时我失去了什么？
我被迫违背了什么？
我为什么不想再回到那种状态？
```

**C. 嫉妒对象**

```text
我会嫉妒哪些人的生活？
我嫉妒的是收入、声望、自由、关系、创造力，还是稳定？
如果去掉社交媒体滤镜，我真正想要其中哪一部分？
```

**D. 愤怒触发器**

```text
哪些事情会让我异常愤怒？
是被控制、被欺骗、低质量沟通、无意义加班、权力羞辱，还是浪费生命？
这个愤怒背后保护的价值是什么？
```

从答案中提取 5 个词。

常见词包括：自由、创造、掌控、健康、家庭、真实、成长、审美、影响力、安全、探索、深度、连接、贡献。

#### 把价值观变成可执行规则

不要写：

```text
我重视自由。
```

要写：

```text
价值观：自由
行为规则：
- 每周至少 3 个半天不被会议占用。
- 不接受需要每日站会和实时在线的“伪自由职业”项目。
- 年度固定安排 2 次 7 天以上的异地生活实验。
- 保持 12 个月生活费安全垫，避免被现金流绑架。
```

再比如：

```text
价值观：健康
行为规则：
- 每天 23:30 前上床，连续 3 天破坏就减少工作量。
- 每周 4 次 30 分钟运动，低于 2 次则下周不新增项目。
- 连续 14 天早醒/失眠/无愉悦感，预约专业帮助。
```

完成标准：每个价值观至少有 3 条可以观察的行为规则。

### 2.4 身份稳定性的压力测试

每月做一次压力测试。

在 `identity.md` 回答：

```text
如果下个月没有新客户，我是谁？
如果前同事升职加薪，我是谁？
如果父母说我“不务正业”，我是谁？
如果一个项目失败，我是谁？
如果收入低于上份工资，我是谁？
```

参考回答结构：

```text
事件：下个月没有新客户。
旧反应：我失败了，我不适合自由职业。
新身份：我是一个正在建设独立现金流系统的人，单月波动是系统输入，不是人格判决。
下一步：复盘获客漏斗，联系 10 个潜在客户，降低本月支出，检查安全垫。
```

完成标准：每个回答都包含“身份不变 + 行动调整”。

## 三、孤独与归属：自由不是一个人消失

远程和独立工作最常见的心理成本不是懒，而是孤独。

孤独有 4 种类型：

| 类型 | 典型表现 | 需要的解法 |
|---|---|---|
| 身体孤独 | 一天没和真人说话 | 线下接触、运动、咖啡馆、共享办公 |
| 专业孤独 | 没人讨论技术/产品决策 | 同行社群、mastermind、代码/产品 review |
| 情绪孤独 | 好消息坏消息无人承接 | 深度朋友、伴侣、咨询师、固定倾诉窗口 |
| 身份孤独 | 身边没人理解你的选择 | 同路人、创业者/自由职业者圈层、长期同行 |

不要用一种社交解决所有孤独。

微信群可以缓解信息孤独，但不一定缓解情绪孤独。

咖啡馆可以缓解身体孤独，但不一定提供专业反馈。

伴侣可以支持你，但不应该承担你全部的职业归属需求。

### 3.1 建立连接组合，而不是等心情好再约人

你需要一个连接组合：

```text
连接组合 = 弱连接 + 同行连接 + 深连接 + 家庭连接 + 公共空间连接
```

建议最低配置：

| 连接类型 | 频率 | 例子 | 完成标准 |
|---|---:|---|---|
| 弱连接 | 每周 2 次 | 社群发言、评论、行业活动 | 有轻量互动 |
| 同行连接 | 每周 1 次 | 技术/产品/独立开发者讨论 | 带走一个反馈 |
| 深连接 | 每周 1 次 | 老友、伴侣、可信任同伴 | 说出真实状态 |
| 家庭连接 | 每周/双周 1 次 | 父母、兄弟姐妹 | 不只报喜报平安 |
| 公共空间连接 | 每周 2–4 次 | 健身房、图书馆、咖啡馆、共享办公 | 身体出现在人群中 |

### 3.2 可执行的每周连接计划

在 `connection-plan.md` 建一个表。

```markdown
# Weekly Connection Plan

| 星期 | 连接动作 | 对象 | 时长 | 目的 | 完成 |
|---|---|---|---:|---|---|
| 周一 | coworking/咖啡馆工作 | 公共空间 | 3h | 身体进入人群 | [ ] |
| 周二 | 给 2 位同行发近况 | 同行 | 20m | 维持弱连接 | [ ] |
| 周三 | mastermind 周会 | 3–5 人小组 | 60m | 专业反馈 | [ ] |
| 周四 | 运动课/跑团 | 线下群体 | 60m | 身体与社交 | [ ] |
| 周五 | 老友语音 | 深连接 | 45m | 真实表达 | [ ] |
| 周六 | 家人电话 | 家庭 | 30m | 降低误解 | [ ] |
| 周日 | 下周约人 | 所有人 | 20m | 提前排程 | [ ] |
```

执行规则：

- 周日晚上先预约，而不是当天临时找人。
- 每周至少一次线下公共空间工作。
- 每周至少一次“可以说真话”的深连接。
- 每周至少一次同行反馈。
- 如果连续两周完成率低于 50%，说明计划太重，先减半。

### 3.3 社群选择清单：不要加入 20 个群，只加入 3 个有效场域

一个有效社群要满足 5 个条件：

- 有明确主题：独立开发、AI 工程、远程工作、投资、写作、产品等。
- 有稳定活动：周会、分享、线下聚会、共学、demo day。
- 有中高质量成员：成员能产出、能反馈、能互助。
- 有低噪音规则：少广告、少情绪倾倒、少无效争论。
- 有现实连接机会：能一对一约聊或线下见面。

评分表：

| 指标 | 0 分 | 1 分 | 2 分 |
|---|---|---|---|
| 主题清晰 | 什么都聊 | 大致相关 | 非常聚焦 |
| 活动稳定 | 无活动 | 偶尔活动 | 固定活动 |
| 成员质量 | 多广告 | 有少数高质量 | 多数能产出 |
| 互动深度 | 只转发 | 有问答 | 有反馈/合作 |
| 线下机会 | 无 | 偶尔 | 经常 |

加入规则：

```text
总分 < 5：退出或静音。
5–7：观察 30 天。
8–10：主动贡献，约 1 对 1。
```

### 3.4 如何发起不尴尬的一对一连接

模板 1：同行学习型

```text
Hi，我最近在做独立接单/AI workflow/远程协作方向的转型。
看到你之前分享过关于 X 的经验，很有启发。
想约你 30 分钟线上聊一下，主要请教 2 个问题：
1. 你当时是怎么处理 Y 的？
2. 如果现在重新来，你会避开什么坑？
我可以提前发问题，也很愿意分享我在 Z 上的经验作为交换。
你这两周哪天方便？
```

模板 2：低压力近况型

```text
好久没聊了。
我最近从坐班转到更独立的工作模式，发现很多东西要重建。
想约你喝咖啡/语音 40 分钟，不求解决问题，就是同步一下近况。
你下周二或周四晚上方便吗？
```

模板 3：合作探索型

```text
我最近在服务 X 类型客户，发现他们普遍有 Y 问题。
你在 Z 领域经验很多，我想听听你的判断：这个问题是否真实、是否有人愿意付费解决。
如果你感兴趣，我们可以 30 分钟交换一下观察，不预设合作。
```

### 3.5 Mastermind 小组：给自由职业者一个“轻量董事会”

Mastermind 是 3–6 人的固定小组，每周或双周一次，彼此汇报目标、卡点、承诺和复盘。

它不是闲聊群。

它的价值是：

- 提供稳定的外部反馈。
- 降低一个人钻牛角尖的概率。
- 让你每周有一次“被看见”的时刻。
- 把模糊焦虑转成具体行动。

#### 60 分钟会议结构

```text
00:00–05:00  开场：每人一句当前状态
05:00–20:00  成员 A：本周进展、卡点、下周承诺
20:00–35:00  成员 B：同上
35:00–50:00  成员 C：同上
50:00–57:00  快速互助：资源、介绍、反馈
57:00–60:00  每人说出下周一个可验证承诺
```

#### 每人汇报模板

```text
上周承诺：
完成情况：完成 / 部分完成 / 未完成
具体证据：链接、截图、数字、客户反馈
本周最大卡点：
我需要大家帮我判断的问题：
下周承诺：
如果我没完成，我要减少/停止什么：
```

#### 组规建议

- 成员数 3–5 人最好。
- 至少连续运行 8 周再评估。
- 不做鸡汤式鼓励，必须给具体反馈。
- 不抢救所有人，重点帮助愿意行动的人。
- 不把小组变成广告和资源置换群。

## 四、中国语境下的外部压力：如何向父母、亲友和社会解释

在中国语境里，“稳定”不只是职业选择。

它常常绑定了：

- 父母的安全感。
- 亲友的面子系统。
- 婚恋市场的评价。
- 户口、房贷、社保、公积金等现实利益。
- “正经工作”的文化想象。

所以当你说“我不坐班了”“我自由职业”“我远程接海外项目”“我做独立产品”，别人听到的可能不是自由，而是：

```text
失业、不稳定、被骗、没前途、不交社保、以后怎么办、是不是混不下去了。
```

你不需要说服所有人。

但你需要准备一套能降低误解、降低冲突、保护边界的表达系统。

### 4.1 先区分三类听众

| 听众 | 目标 | 不要做 | 推荐策略 |
|---|---|---|---|
| 父母/伴侣 | 建立安全感 | 用理念压人 | 讲计划、现金流、边界、备选方案 |
| 亲友/同学 | 降低误解 | 过度解释 | 简短版本，稳定措辞 |
| 不重要的人 | 节省精力 | 辩论人生 | 礼貌结束，不进入细节 |

### 4.2 解释路径：不要讲“我讨厌上班”，要讲“我有过渡方案”

不推荐：

```text
我就是不想给资本家打工。
我受不了坐班了。
我想自由一点。
你们不懂互联网。
```

推荐：

```text
我不是裸辞摆烂，而是在做一个有时间边界的职业转型。
我已经准备了 X 个月生活费安全垫，接下来 Y 个月会验证远程项目/自由职业/独立产品。
如果到某个时间点指标不达标，我会启动备选方案，包括回到全职或选择更稳定的远程岗位。
```

这段话包含 5 个要素：

1. 不是冲动。
2. 有安全垫。
3. 有时间盒。
4. 有验证指标。
5. 有备选方案。

它比“我想自由”更能让家人安心。

### 4.3 父母沟通脚本：第一版，降低焦虑

场景：父母第一次知道你不再坐班。

```text
爸妈，我知道你们听到我不去公司坐班，会担心我是不是失业、是不是不稳定。
我先说结论：我不是没有计划地辞职，也不是不工作。
我现在做的是远程/独立项目，收入来源会从单一工资变成项目收入和长期产品收入。

我已经做了三个准备：
第一，我留了大约 X 个月生活费，不需要你们经济支持。
第二，我给自己设了 Y 个月验证期，每个月看收入、客户、作品和健康状态。
第三，如果到 Y 月还没有达到最低标准，我会考虑回到全职或接更稳定的长期远程合同。

你们可以担心，但不用替我扛这个风险。
我会每个月和你们同步一次进展，用数字说话。
```

可替换数字：

- `X`：建议至少 6，理想 12–24。参见 [应急资金与现金流管理](../investing/04-emergency-fund-cashflow.md)。
- `Y`：建议 6 或 12，不建议无限期。

### 4.4 父母追问：“这算有工作吗？”

父母：

```text
你这个到底算不算有工作？别人问起来我怎么说？
```

回答：

```text
算有工作，只是工作形式不是每天去固定办公室。
以前我是把时间卖给一家公司，现在我是把专业能力卖给多个客户/项目。
你可以简单说：我在做远程软件开发/技术顾问/独立产品，主要服务企业客户。
如果别人问公司，你就说我现在是独立顾问，不在单一公司坐班。
```

如果父母在意面子，可以给他们一句“对外版本”：

```text
他现在做远程技术顾问，主要接企业软件和 AI 自动化项目，比以前更灵活，但也有自己的客户和计划。
```

### 4.5 父母追问：“社保怎么办？以后养老怎么办？”

回答结构：先承认问题，再讲安排。

```text
这个问题很现实，我也不会忽略。
我现在的安排是：
1. 社保/医保：我会通过灵活就业或当地政策继续缴纳，确保医保不断。
2. 商业保险：我会检查重疾、医疗、意外险是否需要补充。
3. 养老：我不会只靠社保，会用长期投资和现金流规划补充。
4. 应急金：我会保留 X 个月生活费，不把所有钱投到高风险项目。

这些我会单独列一个表，不是走一步看一步。
```

注意：涉及社保、税务、保险和投资时，请咨询专业人士。本章只提供沟通框架。

### 4.6 父母追问：“你是不是混不下去了？”

不要急着防御。

可以这样说：

```text
我理解你们会这样担心，因为在你们的经验里，离开单位通常意味着不稳定。
但我这次不是被迫离开，而是主动调整职业模式。
我之前的能力、项目经验和客户需求都还在，只是交付方式变了。

你们可以用结果监督我：
- 每个月是否有收入或有效客户线索；
- 是否持续产出作品；
- 是否保持健康作息；
- 是否按计划复盘。

如果这些都没有，我也会承认问题，而不是硬撑。
```

### 4.7 亲友饭局 15 秒版本

场景：亲戚问“你现在在哪上班？”

版本 A：稳健型

```text
我现在不固定去办公室，主要做远程软件项目和技术顾问，服务企业客户。工作还挺忙，只是形式更灵活。
```

版本 B：简短型

```text
我现在做远程开发，相当于技术顾问，不是传统坐班。
```

版本 C：避免追问型

```text
还在做软件和 AI 相关，只是项目制。最近主要在忙交付，等稳定了再详细说。
```

### 4.8 边界设置：当对方反复贬低你的选择

边界不是吵架。

边界是告诉对方：什么话题可以谈，什么方式不能继续。

#### 轻边界

```text
我知道你是担心我。
这个选择我已经做了计划，我们可以讨论具体风险，但不要用“你就是不稳定/不正经”这种标签来定义我。
```

#### 中边界

```text
如果你愿意听我的计划，我可以讲。
如果只是反复说我不靠谱，那这个话题今天先不聊。
```

#### 强边界

```text
我尊重你的看法，但我的职业选择由我负责。
接下来如果继续用羞辱或否定的方式谈这个话题，我会结束这次聊天。
```

#### 结束对话

```text
我们现在都比较激动，继续聊只会伤感情。
我先挂/先回房间/先出去走走，过几天我把计划写给你看。
```

### 4.9 给伴侣的版本：从“我要自由”改成“我们如何安全过渡”

如果你有伴侣，不能只宣布理想。

你需要讲共同风险。

模板：

```text
我想把职业模式从坐班切到更独立的远程/项目制。
这不是只影响我，也会影响我们的现金流、作息、城市选择和未来计划。

我希望我们一起确认 5 件事：
1. 最低家庭现金流是多少；
2. 我可以用多久验证这个方向；
3. 哪些支出要先冻结；
4. 什么指标说明需要调整；
5. 我在家工作时，空间和时间边界怎么安排。

如果你有担心，我们先列出来，不急着争对错。
```

伴侣沟通完成标准：

- 双方知道每月最低支出。
- 双方知道安全垫能撑多久。
- 双方知道验证期结束日期。
- 双方知道失败预案。
- 双方知道工作时间和家庭时间边界。

## 五、恐惧、不确定性与比较：把情绪变成可处理的数据

自由生活中的恐惧通常不是单一恐惧。

它由 5 类输入组成：

| 恐惧 | 典型念头 | 可处理变量 |
|---|---|---|
| 收入波动 | 下个月没钱怎么办 | 安全垫、获客漏斗、支出结构 |
| 身份下滑 | 我是不是不如前同事 | 身份系统、作品证据、比较对象 |
| 能力怀疑 | 我是不是没有公司就不行 | 案例、反馈、复盘、技能栈 |
| 未来失控 | 这样下去会不会完蛋 | 时间盒、预案、指标 |
| 社会评价 | 别人会不会觉得我失败 | 沟通边界、价值观、圈层 |

关键原则：

> 不要试图消灭恐惧。把恐惧拆成变量，然后逐个降低风险。

### 5.1 “焦虑工单”法：像处理 bug 一样处理情绪

在 `fear-log.md` 建一个表。

```markdown
| 日期 | 触发事件 | 自动念头 | 情绪强度 0-10 | 证据支持 | 证据反驳 | 更平衡的想法 | 下一步行动 |
|---|---|---|---:|---|---|---|---|
```

示例：

```markdown
| 日期 | 触发事件 | 自动念头 | 情绪强度 | 证据支持 | 证据反驳 | 更平衡的想法 | 下一步行动 |
|---|---|---|---:|---|---|---|---|
| 7/7 | 看到前同事升职 | 我离开公司就是失败 | 8 | 他 title 更高、收入可能更高 | 我有 12 个月安全垫、正在建立客户管道、我选择的是不同游戏 | 他在公司路径上前进，不等于我在自由路径上失败 | 今天写一篇案例，联系 3 个潜在客户 |
```

完成标准：情绪强度从记录前到记录后至少下降 1–2 分，或至少产出一个行动。

### 5.2 常见认知扭曲与重构模板

| 认知扭曲 | 原始念头 | 重构问题 | 平衡说法 |
|---|---|---|---|
| 灾难化 | 这个月收入低，以后都完了 | 最坏情况是什么？概率多大？预案是什么？ | 单月波动是信号，不是判决 |
| 非黑即白 | 没超过上份工资就是失败 | 成功是否只有一个指标？ | 过渡期同时看收入、自由度、健康和资产 |
| 读心术 | 亲戚一定觉得我没出息 | 我有证据吗？他们的评价重要吗？ | 我可以准备简短解释，不必赢得所有人认可 |
| 应该句式 | 我这个年纪应该稳定了 | 这个“应该”来自谁？适合我吗？ | 我需要的是可持续，而不是符合模板 |
| 选择性注意 | 只看到别人融资/升职 | 我忽略了哪些成本和风险？ | 社交媒体展示的是高光，不是全账本 |
| 人格化 | 项目没成说明我不行 | 还有哪些非人格因素？ | 项目失败需要复盘，不需要羞辱自己 |

### 5.3 收入波动焦虑：用安全垫换回神经系统稳定

自由职业者最容易犯的错是：只算收入，不算神经系统。

如果你只有 1–2 个月生活费，每次客户延迟付款都会变成生存威胁。

这时你会：

- 接不该接的项目。
- 报价过低。
- 对客户边界过软。
- 无法深度工作。
- 对伴侣和家人更易怒。

所以应急资金不是保守，而是自由的基础设施。

请结合 [应急资金与现金流管理](../investing/04-emergency-fund-cashflow.md) 建立自己的安全垫。

也请回看 [安全退出与过渡期设计](01-safe-exit-transition.md)：退出不是一个日期，而是一段现金流、能力、客户和心理状态共同达标的过渡。

#### 安全垫计算公式

```text
月基础支出 = 房租/房贷 + 食物 + 交通 + 医疗保险 + 通讯 + 家庭责任 + 最低娱乐 + 税费预留
最低安全垫 = 月基础支出 × 6
推荐安全垫 = 月基础支出 × 12
高不确定性安全垫 = 月基础支出 × 18–24
```

示例：

```text
房租：6000
食物：3000
保险/医疗：1000
交通通讯：800
家庭责任：2000
其他基础支出：2200
月基础支出：15000

最低安全垫：15000 × 6 = 90000
推荐安全垫：15000 × 12 = 180000
高不确定性安全垫：15000 × 18 = 270000
```

把这个数字写进 `fear-log.md` 顶部。

```text
我的月基础支出：
我的当前安全垫：
可覆盖月数：
低于几个月时必须降低风险：
低于几个月时必须启动全职/长期合同预案：
```

建议阈值：

- 12 个月以上：可以探索，但仍要有节奏。
- 6–12 个月：可以运行，但每月必须跟踪获客与支出。
- 3–6 个月：优先现金流，不做高风险长期赌注。
- 3 个月以下：立即降低支出，寻找稳定合同或全职选项。

### 5.4 冒名顶替感：用证据库替代自我审判

冒名顶替感常见于：

- 第一次独立报价。
- 第一次对外称“顾问”。
- 第一次卖自己的产品。
- 第一次没有公司背书去谈客户。
- 第一次面对比你更有名的人。

解决办法不是反复告诉自己“我很棒”。

解决办法是建立证据库。

在 `identity.md` 或单独 `proof.md` 写：

```markdown
# Proof Bank

## 我解决过的问题
- 问题：
  背景：
  我的动作：
  结果：
  可复用能力：

## 别人给过的正反馈
- 原话：
  来源：
  对应能力：

## 可展示作品
- 链接：
  解决什么问题：
  证明什么能力：

## 我能稳定交付的任务
- 任务类型：
  交付周期：
  质量标准：
  风险边界：
```

当你不敢报价时，不要问“我配不配”。

问：

```text
我是否解决过类似问题？
我是否能定义范围和交付标准？
我是否能承担这个项目的风险？
客户是否能从结果中获得高于价格的价值？
```

如果答案是“是”，你就不是冒充者。

你是在承担一个可定义的专业服务。

### 5.5 社交媒体比较：建立信息摄入协议

社交媒体会让自由职业者误判世界。

你看到的是：

- 别人月入 10 万。
- 别人产品爆了。
- 别人旅居很爽。
- 别人融资了。
- 别人英语流利、客户稳定、身材很好、伴侣支持。

你看不到的是：

- 现金流波动。
- 税务和合同麻烦。
- 家庭冲突。
- 健康代价。
- 长期试错。
- 背后的资产、资源和运气。

建立一个简单协议：

```text
每天社交媒体总时长：≤ 30 分钟
睡前 1 小时：不刷职业/收入相关内容
看到刺激内容后：必须写一行“我不知道的成本是什么”
每周一次：取关/静音让我持续自卑但无行动价值的账号
每月一次：列出 3 个真正可学习对象，而不是情绪触发对象
```

替代动作：

- 把“刷别人成功”换成“分析一个商业案例”。
- 把“羡慕别人作品”换成“给自己的作品加一个 demo”。
- 把“看别人旅居”换成“做一次 3 天低成本异地工作实验”。

## 六、目标与动机：自由是为了什么

很多人成功逃离坐班后，会进入第二层危机：

```text
我终于不用上班了，然后呢？
```

如果自由只被定义为“不要什么”，你会很快空掉。

你需要定义“自由是为了什么”。

### 6.1 四种自由，不要只追一种

| 自由类型 | 问题 | 例子 |
|---|---|---|
| 时间自由 | 我能决定什么时候工作吗？ | 上午深度工作，下午运动陪家人 |
| 地点自由 | 我能决定在哪里生活吗？ | 在不同城市/国家短住 |
| 财务自由 | 我能拒绝不合适的工作吗？ | 有安全垫和投资现金流 |
| 精神自由 | 我能按价值观生活吗？ | 不为了面子接消耗型机会 |

一个常见陷阱是：只追地点自由。

结果变成：人在海边，心在 Slack；人在清迈，焦虑国内客户；人在旅居，天天赶 deadline。

真正的自由要同时看四个维度。

这也是本书总纲 [《逃离坐班白皮书》](../00-manifesto.md) 反复强调的：自由不是逃避劳动，而是重新掌控技能、市场、变现和资本的组合。

### 6.2 目的地图：从“逃离”切换到“建设”

在 `purpose-map.md` 写下 5 个问题。

```text
1. 如果我不再用公司晋升衡量人生，我用什么衡量？
2. 我希望未来 3 年普通的一周长什么样？
3. 我愿意服务哪类人/问题 10 年？
4. 我希望自由给我的关系带来什么改善？
5. 我希望自由给我的健康、创造和学习带来什么空间？
```

然后写一段“自由用途声明”。

模板：

```text
我追求自由，不是为了永远不工作，
而是为了把更多时间投入到：
- 我愿意长期解决的问题：
- 我想建立的关系质量：
- 我想保护的健康状态：
- 我想创造的作品/产品/服务：
- 我想体验的生活方式：

因此，我会拒绝：
- 
我会主动选择：
- 
```

示例：

```text
我追求自由，不是为了永远不工作，
而是为了把更多时间投入到：
- 帮中小团队把 AI 真正落地到业务流程；
- 每周至少 4 晚和家人正常吃饭；
- 保持睡眠、运动和长期学习；
- 建立可复用的软件产品和公开作品；
- 每年在 2–3 个城市短住，理解不同生活方式。

因此，我会拒绝：需要长期即时响应、不断救火、没有边界的项目。
我会主动选择：能沉淀作品、复利技能、支持健康节奏的客户和产品。
```

### 6.3 动机可持续：不要只靠兴奋感

自由生活早期常见动机曲线：

```text
第 1–2 周：兴奋，睡到自然醒，疯狂规划
第 3–6 周：拖延，节奏混乱，开始怀疑
第 2–3 月：收入压力出现，开始焦虑
第 4–6 月：要么建立系统，要么回到旧模式
```

所以动机不能只靠“我终于自由了”。

你需要 4 种燃料：

| 燃料 | 作用 | 例子 |
|---|---|---|
| 价值燃料 | 知道为什么做 | 自由用途声明 |
| 进展燃料 | 看见自己在推进 | 每周复盘、作品发布 |
| 连接燃料 | 被同行看见 | mastermind、社群 demo |
| 恢复燃料 | 身体不崩 | 睡眠、运动、休息日 |

如果你只有价值燃料，没有进展，你会理想化。

如果你只有进展燃料，没有恢复，你会燃尽。

如果你只有连接燃料，没有价值，你会被别人节奏带走。

### 6.4 每季度重新定义一次“足够好”

坐班系统里，外部会定义“足够好”：绩效、晋升、调薪、title。

独立生活里，你要自己定义。

季度模板：

```text
本季度我定义的足够好：
1. 财务：最低收入/现金流目标是：
2. 作品：至少发布/交付：
3. 健康：睡眠、运动、体检、情绪目标是：
4. 关系：每周/每月固定连接是：
5. 学习：要掌握或升级的能力是：
6. 自由：我要使用自由做的一件具体事是：
```

示例：

```text
本季度足够好：
- 财务：月均收入达到基础支出的 1.2 倍；
- 作品：发布 2 篇案例文章，交付 1 个客户项目；
- 健康：每周运动 3 次，平均睡眠 7 小时；
- 关系：每周 1 次深连接，每两周 1 次同行会；
- 学习：完成一个 LLM eval 小项目；
- 自由：安排 5 天不工作旅行，不带电脑处理客户需求。
```

完成标准：季度目标不会把你重新变成“没有老板的加班机器”。

## 七、心理健康护栏：一个人工作时，风险更隐蔽

独立工作者的心理风险常常来得很安静。

没人会在办公室看见你连续几天状态不对。

没人会提醒你午饭没吃。

没人会发现你已经两周没有出门。

所以你需要自检仪表盘。

### 7.1 绿色、黄色、红色状态

| 状态 | 信号 | 行动 |
|---|---|---|
| 绿色 | 睡眠稳定、有食欲、能工作、能社交、能感到愉悦 | 保持当前节奏 |
| 黄色 | 连续 3–7 天失眠/拖延/易怒/回避/暴食或没胃口 | 降低工作量，增加连接，复盘压力源 |
| 红色 | 连续 2 周明显低落、无望、无法工作、自伤念头、惊恐发作 | 立即寻求专业帮助和可信任支持 |

### 7.2 每周自检评分卡

在 `mental-health-guardrails.md` 建表。

```markdown
# Weekly Self Check

| 指标 | 0 分 | 1 分 | 2 分 | 本周分数 |
|---|---|---|---|---:|
| 睡眠 | <5h 或严重失眠 | 5–6.5h/不稳定 | 7h 左右且稳定 | |
| 饮食 | 经常忘吃/暴食 | 勉强规律 | 基本规律 | |
| 运动 | 0 次 | 1–2 次 | ≥3 次 | |
| 深度工作 | 几乎没有 | 有但零散 | ≥3 个高质量时段 | |
| 社交连接 | 0 次真实交流 | 1 次 | ≥2 次 | |
| 情绪 | 持续低落/易怒 | 波动但可恢复 | 基本稳定 | |
| 愉悦感 | 没有任何期待 | 偶尔有 | 有明确愉悦来源 | |
| 边界 | 工作侵占全部生活 | 偶尔失控 | 有停止工作时间 | |
```

解释：

```text
总分 12–16：绿色，保持。
总分 8–11：黄色，下周减少 20% 工作量，增加连接和运动。
总分 0–7：红色，联系专业帮助或可信任的人，不做重大职业决定。
```

### 7.3 黄色状态干预清单

当你进入黄色状态，先不要立刻做人生决策。

执行 72 小时稳定协议：

```text
第 1 天：
- 取消非必要会议和新承诺。
- 吃两顿正常饭。
- 出门晒太阳或走路 30 分钟。
- 给一个可信任的人发：“我最近状态不太好，想和你聊 20 分钟。”

第 2 天：
- 睡前 1 小时不看工作和社交媒体。
- 写下当前压力源，分成可控/不可控。
- 完成一个 25 分钟最小工作块，不追求效率。
- 检查是否连续多日咖啡因、酒精或熬夜过量。

第 3 天：
- 复盘现金流和项目，而不是在脑内灾难化。
- 与同行或 mastermind 同步真实状态。
- 如果仍然明显恶化，预约专业咨询或就医。
```

### 7.4 红色状态：立即升级支持

红色信号包括但不限于：

- 连续两周以上几乎每天情绪低落或空洞。
- 对过去喜欢的事失去兴趣。
- 明显睡眠问题：严重失眠、早醒、嗜睡。
- 食欲和体重明显变化。
- 无法完成基本工作或生活任务。
- 强烈无望感、羞耻感、自责。
- 惊恐发作、呼吸困难、濒死感。
- 自伤或轻生念头。

行动顺序：

1. 如果有自伤或轻生风险，立即联系当地紧急服务、危机热线或可信任的人陪同就医。
2. 不要独自做重大决定：辞职、搬家、分手、卖资产、签大合同。
3. 暂停高风险项目和高压客户沟通。
4. 告诉至少一个现实中的人：“我现在需要支持，不只是建议。”
5. 尽快预约精神科/心理科或持证心理咨询。

一句重要的话：

> 需要帮助不是自由失败，而是系统在保护你继续活下去、继续选择。

## 八、关系与家庭物流：自由生活不是只由你一个人承担

自由职业、远程工作、数字游民和独立产品会改变家庭系统。

它影响的不只是你：

- 伴侣是否接受收入波动。
- 家庭是否能适应你在家工作。
- 父母是否需要照护。
- 孩子教育和作息是否稳定。
- 居住城市和签证/租房/医疗是否匹配。
- 两个人对风险的承受能力是否不同。

### 8.1 家庭风险会议：每月一次，不在吵架时开

建议每月固定一次 60–90 分钟。

议程：

```text
1. 现金流：本月收入、支出、安全垫变化。
2. 工作量：是否过载，是否影响家庭时间。
3. 空间：在家工作是否打扰彼此。
4. 关系：本月是否有被忽略/被控制/不安全感。
5. 下月计划：项目、旅行、家庭安排、风险点。
6. 决策：需要共同确认的支出、搬迁、合同、休假。
```

规则：

- 不在深夜开。
- 不在刚吵完架时开。
- 不用“你总是/你从来”开头。
- 每次只解决 1–2 个关键问题。
- 会议结束要有行动项。

### 8.2 伴侣风险偏好对齐表

```markdown
| 问题 | 我的答案 | 伴侣答案 | 差异 | 对齐动作 |
|---|---|---|---|---|
| 最低安全垫要几个月？ | | | | |
| 月收入低于多少会焦虑？ | | | | |
| 可以接受多久验证期？ | | | | |
| 是否接受异地/旅居？ | | | | |
| 是否愿意降低消费？ | | | | |
| 是否希望我回全职？触发条件是什么？ | | | | |
```

完成标准：差异不是被压下去，而是被写出来。

### 8.3 在家工作边界协议

如果你和家人同住，必须定义边界。

模板：

```text
我的深度工作时间：周一到周五 9:00–12:00 / 14:00–17:00
这段时间如果门关着/戴耳机，默认不打扰。
可以打扰的情况：孩子安全、紧急电话、家里突发事件。
不建议打扰的情况：顺手问一句、临时家务、亲友闲聊。

我的家庭时间：18:30–21:30
这段时间我不主动处理客户消息，除非提前说明有紧急交付。

每周休息时间：周六下午 + 周日
如果需要加班，提前 24 小时沟通，并安排补偿休息。
```

这不是把家人当员工管理。

这是把模糊期待变成可协商规则。

### 8.4 旅居/搬迁前的关系清单

如果你计划数字游民或跨城市生活，先回答：

```text
1. 这次移动是探索、降成本、陪伴家人，还是逃避当前问题？
2. 伴侣/家人是否真正同意，还是只是“不反对”？
3. 医疗、保险、社保、税务、签证是否有基础安排？
4. 工作网络、时区、客户沟通是否可运行？
5. 到达后每周社交如何建立？
6. 如果 30 天后状态变差，撤退方案是什么？
```

旅居不是治疗焦虑的万能药。

如果你的问题是现金流、边界和意义感，换城市只能暂时提供新鲜感。

## 九、日常实践：把心理可持续性放进日程

心态不是想出来的。

心态是被每天的睡眠、运动、现金流、作品、连接和边界训练出来的。

### 9.1 每日 20 分钟稳定仪式

早上或开始工作前执行。

```text
1. 身体扫描 2 分钟：我睡得如何？哪里紧？是否饿/渴？
2. 情绪命名 2 分钟：我现在最强的情绪是什么？焦虑/兴奋/疲惫/抗拒？
3. 今日三件事 5 分钟：只写 3 件最重要的事。
4. 恐惧拆解 5 分钟：今天最担心什么？下一步可控动作是什么？
5. 连接动作 3 分钟：今天要联系谁？发哪一句话？
6. 停止工作时间 3 分钟：今天几点收工？收工后做什么恢复？
```

记录模板：

```markdown
## YYYY-MM-DD

睡眠：
身体：
情绪：
今日三件事：
1.
2.
3.
今日担心：
可控下一步：
连接动作：
收工时间：
```

完成标准：每天不超过 20 分钟，不变成新的拖延仪式。

### 9.2 每周 45 分钟复盘

建议周五下午或周日晚上。

```text
1. 本周完成了什么：列证据，不列感受。
2. 本周哪里过载：项目、关系、身体、信息摄入。
3. 本周现金流状态：收入、支出、应收款、安全垫月数。
4. 本周连接状态：和谁有真实交流，是否孤独。
5. 本周身份状态：是否过度依赖外部评价。
6. 下周一个主题：不要超过一个。
7. 下周要停止什么：停止比新增更重要。
```

表格模板：

```markdown
| 维度 | 本周事实 | 评分 1-5 | 下周调整 |
|---|---|---:|---|
| 现金流 | | | |
| 作品/交付 | | | |
| 获客/市场 | | | |
| 睡眠/运动 | | | |
| 关系连接 | | | |
| 情绪稳定 | | | |
| 自由使用 | | | |
```

### 9.3 每月一次“自由审计”

自由审计不是问“我赚了多少钱”。

它问：我有没有用好自由。

```text
本月我因为自由多做了什么？
- 多陪伴了谁？
- 创造了什么？
- 学到了什么？
- 保护了什么健康边界？
- 拒绝了什么不适合的机会？

本月我把自由浪费在了什么？
- 无意识刷手机？
- 无边界接活？
- 逃避困难对话？
- 过度计划但不交付？

下月我要让自由服务哪一个具体目标？
```

完成标准：每月至少有一个“自由被使用”的证据。

## 十、自检评分卡：你现在是否适合继续独立

每月最后一天打分。

| 维度 | 1 分 | 3 分 | 5 分 | 当前分 |
|---|---|---|---|---:|
| 现金流 | 低于 3 个月安全垫 | 6–12 个月安全垫 | 12+ 个月或稳定合同 | |
| 获客 | 没有线索 | 有零散线索 | 有稳定管道 | |
| 交付 | 经常拖延失控 | 基本能交付 | 稳定高质量交付 | |
| 身份 | 依赖 title/比较 | 偶尔波动 | 价值和作品导向 | |
| 连接 | 基本孤立 | 有少量连接 | 稳定社群/深连接 | |
| 健康 | 睡眠情绪明显恶化 | 可维持 | 稳定恢复系统 | |
| 家庭关系 | 高冲突/隐瞒 | 可沟通 | 有共同计划 | |
| 意义感 | 只想逃离 | 有模糊方向 | 知道自由服务什么 | |

解释：

```text
总分 32–40：可以继续扩大独立生活实验。
总分 24–31：维持当前规模，优先补短板。
总分 16–23：降低风险，增加稳定合同或兼职收入。
总分 <16：暂停扩张，考虑回到更稳定结构，同时修复健康/现金流/关系。
```

不要把低分当失败。

低分是仪表盘报警。

工程师不会因为监控报警就羞辱服务器，也不要因为自己报警就羞辱自己。

## 十一、90 天计划：建设一个可持续的独立生活系统

下面是一份可以直接执行的 90 天计划。

它默认你已经或即将进入远程/自由/独立工作状态。

如果你还在过渡期，请先配合 [安全退出与过渡期设计](01-safe-exit-transition.md) 使用。

### 第 1–30 天：稳定系统，不追求爆发

目标：建立身份、现金流、节奏和连接的最低可运行版本。

#### 第 1 周：盘点现实

- [ ] 写 `identity.md` 三层身份。
- [ ] 写 `values.md` 5 个价值观和行为规则。
- [ ] 计算月基础支出和安全垫月数。
- [ ] 建立每日 20 分钟稳定仪式。
- [ ] 和伴侣/家人说明当前计划。
- [ ] 本周至少一次线下公共空间工作。

完成标准：你知道自己是谁、能撑多久、每天怎么开始。

#### 第 2 周：建立边界

- [ ] 设定固定工作时间和收工时间。
- [ ] 写在家工作边界协议。
- [ ] 清理 3 个最消耗的信息源。
- [ ] 给父母/亲友准备 15 秒解释版本。
- [ ] 约一个同行或老友聊天。
- [ ] 在 [一个人的生产力系统](03-solo-productivity-system.md) 基础上定义每周复盘。

完成标准：你不是全天候待命，也不是随时内疚。

#### 第 3 周：建立连接

- [ ] 选择 1–3 个高质量社群。
- [ ] 发起 2 次一对一约聊。
- [ ] 尝试加入或创建 mastermind。
- [ ] 固定每周一次深连接。
- [ ] 安排一次运动或兴趣类线下活动。

完成标准：你的生活里有稳定的人，而不只是稳定的屏幕。

#### 第 4 周：建立证据

- [ ] 建立 `proof.md` 证据库。
- [ ] 整理 3 个过去项目案例。
- [ ] 发布或更新一个作品/案例页面。
- [ ] 写第一份月度自由审计。
- [ ] 根据自检评分卡补一个最低分维度。

完成标准：你不再只靠感觉证明自己。

### 第 31–60 天：修复短板，开始稳定输出

目标：把心理稳定性和市场行动接起来。

#### 第 5–6 周：现金流与恐惧管理

- [ ] 每周更新一次安全垫月数。
- [ ] 记录至少 5 条焦虑工单。
- [ ] 找出最常见的 2 个焦虑触发器。
- [ ] 为每个触发器写预案。
- [ ] 若安全垫低于 6 个月，优先找稳定合同。

完成标准：焦虑能被记录、讨论、行动，而不是在脑内循环。

#### 第 7–8 周：关系与家庭对齐

- [ ] 开一次家庭风险会议。
- [ ] 填伴侣风险偏好对齐表。
- [ ] 和父母同步一次数字化进展。
- [ ] 复盘是否有隐瞒、逃避或防御性沟通。
- [ ] 安排一次不谈工作的陪伴。

完成标准：亲密关系不是被你的自由生活消耗，而是被纳入设计。

### 第 61–90 天：定义长期方向

目标：从“我终于离开了”转向“我要建设什么”。

#### 第 9–10 周：目的与作品

- [ ] 写 `purpose-map.md`。
- [ ] 写自由用途声明。
- [ ] 选择一个愿意长期服务的问题域。
- [ ] 发布一个能代表你方向的作品、文章、demo 或服务页。
- [ ] 拒绝一个不符合价值观的机会，或调整其边界。

完成标准：你的自由开始指向一个建设性目标。

#### 第 11–12 周：系统复盘

- [ ] 完成 8 维自检评分卡。
- [ ] 对比第 1 周和第 12 周的睡眠、收入、连接、情绪。
- [ ] 决定下一季度策略：扩张、维持、收缩或回到稳定结构。
- [ ] 更新家庭/伴侣沟通版本。
- [ ] 给未来 90 天设定一个主题。

完成标准：你不是靠情绪决定继续或放弃，而是靠数据、价值观和关系共识。

## 十二、常见坑与排查表

| 坑 | 症状 | 根因 | 修复动作 |
|---|---|---|---|
| 把自由等同于不工作 | 昼夜颠倒、拖延、内疚 | 没有目标和节奏 | 建每日仪式和每周复盘 |
| 把自己关在家里 | 一周没见真人，情绪变差 | 连接系统缺失 | 每周 2 次公共空间 + 1 次深连接 |
| 过度向父母解释 | 每次沟通都像答辩 | 想获得完全认可 | 使用短版本 + 月度数字同步 |
| 看到别人成功就崩 | 刷社媒后焦虑 | 比较对象不完整 | 信息摄入协议 + 焦虑工单 |
| 为现金流接烂项目 | 客户越界、睡眠恶化 | 安全垫不足或边界弱 | 提高安全垫，重写合同边界 |
| 伴侣越来越不支持 | 对方觉得被排除在外 | 风险没有共同讨论 | 每月家庭风险会议 |
| 旅居后更孤独 | 换城市后无人连接 | 把地点当解药 | 抵达前安排社群/运动/共享办公 |
| 目标越来越散 | 什么都想做，什么都不深 | 自由没有用途声明 | 写 purpose-map，季度只选一个主题 |
| 情绪持续低落仍硬撑 | 工作和生活都失控 | 心理健康红灯 | 暂停重大决策，寻求专业帮助 |

## 十三、可复制模板合集

### 13.1 自我介绍模板

```text
我过去主要做 X，现在更聚焦 Y。
我帮助 Z 类型客户解决 A 问题，通常通过 B 方法交付。
最近我在做 C，如果你身边有人遇到类似问题，可以介绍我们聊聊。
```

示例：

```text
我过去做全栈和内部平台，现在更聚焦 AI 自动化和业务流程工具。
我帮助 10–100 人的团队把重复运营、客服和数据流程自动化，通常用 LLM workflow、后台系统和数据看板交付。
最近我在做销售线索自动整理工具，如果你身边有团队被表格和手工流程困住，可以介绍我们聊聊。
```

### 13.2 向父母月度同步模板

```text
这个月同步一下我的情况：
1. 收入：本月收入 X，应收款 Y，基础支出 Z。
2. 安全垫：目前还能覆盖 N 个月生活。
3. 工作：完成了 A 项目，正在谈 B 项目。
4. 健康：作息/运动/状态大致如何。
5. 下月计划：重点做什么。
6. 风险：目前最大的风险是什么，我准备怎么处理。

你们不用替我解决，知道我在认真管理就好。
```

### 13.3 客户边界模板

```text
为了保证交付质量，我的工作方式是：
- 响应时间：工作日 X 小时内回复，非紧急事项不实时响应。
- 会议：每周固定 Y 次，每次 Z 分钟。
- 需求变更：超出原范围的需求会单独评估时间和费用。
- 紧急支持：如果需要晚间/周末支持，需要提前约定费用和范围。
```

这部分也会保护心理健康。

没有边界的客户关系，会把自由职业变成更糟糕的坐班。

### 13.4 低谷求助模板

发给朋友/同行：

```text
我最近状态不太好，不是想听大道理，主要需要有人陪我把情况说清楚。
你这两天方便 30 分钟语音吗？
我想讲三件事：现在的压力源、我已经做了什么、下一步最小动作。
如果你没时间也没关系，告诉我一个你方便的时间即可。
```

发给专业人士预约时：

```text
你好，我最近出现了持续的情绪低落/焦虑/失眠/无法工作等情况，已经持续 X 周。
我目前是自由职业/远程工作状态，压力源主要是收入波动、孤独、家庭沟通和不确定性。
想预约一次评估，了解是否需要咨询或进一步就医。
```

## 十四、练习与里程碑作业

### 初级：一周内完成

- [ ] 写三层身份，不少于 500 字。
- [ ] 提取 5 个价值观，每个价值观 3 条行为规则。
- [ ] 计算安全垫月数。
- [ ] 给父母/亲友准备 15 秒解释版本。
- [ ] 安排一次深连接聊天。

### 中级：30 天内完成

- [ ] 建立每周连接计划并执行 4 周。
- [ ] 记录 10 条焦虑工单。
- [ ] 加入或创建一个 mastermind，并运行至少 4 次。
- [ ] 完成一次家庭风险会议。
- [ ] 发布一个能证明你新身份的作品或案例。

### 高级：90 天内完成

- [ ] 完整执行 90 天计划。
- [ ] 完成 3 次月度自由审计。
- [ ] 建立稳定的同行反馈机制。
- [ ] 把安全垫、获客、健康、关系纳入同一张月度仪表盘。
- [ ] 做出下一季度策略：扩张、维持、收缩或回到稳定结构。

## 十五、最终检查清单

- [ ] 我能用 30 秒说明自己是谁，而不依赖公司和职级。
- [ ] 我知道自己最重要的 5 个价值观，以及对应行为规则。
- [ ] 我知道当前安全垫能覆盖几个月。
- [ ] 我知道安全垫低于哪个阈值时必须降低风险。
- [ ] 我有每周连接计划，而不是等孤独时临时求救。
- [ ] 我至少有一个同行反馈机制。
- [ ] 我准备了给父母、亲友、伴侣的不同解释版本。
- [ ] 我能识别灾难化、非黑即白、读心术等常见认知扭曲。
- [ ] 我有焦虑工单表，可以把恐惧转成行动。
- [ ] 我知道自由是为了什么，而不只是为了逃离什么。
- [ ] 我有每周心理健康自检评分卡。
- [ ] 我知道黄色状态和红色状态分别该做什么。
- [ ] 我与伴侣/家人讨论过现金流、边界和验证期。
- [ ] 我有 90 天建设计划，并知道本周要做什么。
- [ ] 我能在必要时寻求专业帮助，而不是把硬扛当成熟。

## 本章最后一句话

你逃离坐班，不是为了成为一个更孤独、更焦虑、更难被理解的人。

你逃离坐班，是为了把人生从默认配置改成自定义配置。

自定义配置需要工程能力：需求分析、风险控制、监控报警、迭代复盘、故障恢复。

也需要人的能力：连接、表达、求助、爱、边界、意义。

请把自由当成一个长期系统来维护。

它不是一次辞职。

它是一种持续建设。
