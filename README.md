# 逃离坐班白皮书

> 面向资深软件工程师的系统性"逃离路径"：把你的技能、市场、变现、资本四种杠杆叠起来，一步步走出格子间，换回对时间、地点与人生的掌控权。

这不是一本"如何找到更好工作"的书。**换一份工作，只是换一个格子间。** 本书要谈的是：如何用工程师本就具备的稀缺能力，逐级打开时间自由、地点自由、财务自由与精神自由——最终让"坐班"从生存必需变成一个可选项。

全书由 **1 篇总纲 + 6 册白皮书** 组成，纯 Markdown、可离线阅读、GitHub 直读。

---

## 谁适合读

- 有几年扎实工程经验、但被通勤与工位困住的开发者；
- 想把技能接入全球远程/海外市场、拿回地点自由的人；
- 想从"只有工资"走向"多元自主收入"直到财务独立的人；
- 不想再用"考进下一家大厂"定义人生的人。

## 本书的核心主张：自由度模型

"逃离坐班"不是一次性的辞职动作，而是四种自由的逐步累积：

| 自由维度 | 含义 | 主要由哪册支撑 |
| --- | --- | --- |
| **时间自由** | 不再用固定工时换钱，掌控自己的日程 | WP-05 变现 · WP-06 远程生活 |
| **地点自由** | 在哪工作/生活由你决定，而非公司 | WP-02 出海远程 · WP-03 英语 · WP-06 |
| **财务自由** | 被动现金流覆盖生活成本，工作变可选 | WP-04 投资 · WP-05 变现 |
| **精神自由** | 身份不再被职级定义，可持续、不焦虑 | 总纲 · WP-06 心态 |

## 五个逃离杠杆

真正让人离开格子间的，从来不是"努力工作"，而是叠加**杠杆**：

1. **技能杠杆**（护城河）——把 AI 工程做成别人替代不了的能力 → **WP-01**
2. **市场杠杆**（把技能卖到全球高价市场）——远程/出海 + 英语 → **WP-02 · WP-03**
3. **变现杠杆**（从单一工资 → 自主多元收入）——自由职业/独立产品/咨询/内容 → **WP-05**
4. **资本杠杆**（让钱替你工作）——指数化 + 资产配置 + FIRE → **WP-04**
5. **落地与退出**（把上面四条安全地兑现）——安全离职、数字游民、一人公司自律 → **WP-06**

## 四阶段逃离阶梯

> 详细的自我定位测评、路线图与 90 天启动计划见 **[总纲：逃离坐班宣言与自由度模型](00-manifesto.md)**。

1. **第 0 阶·地基**：应急金 + 现金流稳住，建立"跑道"（WP-04 地基章）。
2. **第 1 阶·杠杆**：把技能做深（WP-01）、接入全球市场拿到远程/高薪 offer（WP-02/03）。
3. **第 2 阶·变现**：在职期间用副业跑通第一笔自主收入（WP-05）。
4. **第 3 阶·退出**：副业收入 + 被动现金流覆盖成本后，安全离职、走向远程/独立生活（WP-06 + WP-04）。

## 如何使用本书

- **先读总纲**，用自测找到你所处的阶段，再按阶梯选择当前最该读的一册。
- 每章都尽量做到"可操作"：含模板、清单、数值示例与练习，读完能直接动手。
- 投资/税务/签证/法律相关内容**仅供教育与参考，非专业建议**；请结合自身情况并咨询专业人士。
