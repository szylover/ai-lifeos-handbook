# WP-05 · 独立变现与自由职业白皮书（变现杠杆）

> 从单一工资到自主多元收入：把已有技能直接、可复现地变成钱

**版本** v1.0 · **更新** 2026-07-07

## 摘要

在《逃离坐班白皮书》里，这一册负责**变现杠杆**。逃离坐班的关键转折，是收入不再只来自"一份工资、一个雇主"，而是来自你能自主控制的多个渠道：自由职业、独立产品、咨询、内容。本册系统拆解这些变现模式的机制、启动路径、定价与获客，帮你在**离职之前**就跑通第一笔自主收入。

## 执行摘要

自主收入分两大类：**卖时间的杠杆化**（自由职业、咨询——单价更高、更自由，但仍与投入挂钩）与**与时间解耦**（独立产品、内容——前期投入大，但可规模化、可睡后收入）。理性路径是：先用高时薪的接单/咨询快速建立现金流与"跑道"，同时用一部分精力孵化可解耦的产品或受众资产，逐步把收入结构从"卖时间"迁移到"卖资产"。

本册每一章都力求可操作：给出平台、定价模型、合同要点、获客漏斗、真实数字算例与可套用模板，让你读完能立刻迈出第一步。

> 本册涉及收入、税务、合同处仅供教育与参考，非专业法律/税务建议。
